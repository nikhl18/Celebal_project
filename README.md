# Celebal_project
To address RetailMart’s fragmented data challenges, a centralized data analytics platform is designed to integrate, clean, and transform data from multiple raw sources into a structured and reliable system. The solution follows the Medallion Architecture, organizing data into Bronze (raw), Silver (cleaned), and Gold (business-ready) layers to ensure scalability, data quality, and efficient processing. This end-to-end solution enables RetailMart to generate accurate, real-time insights, empowering business teams to make data-driven decisions, track key performance indicators, and improve overall operational efficiency.
